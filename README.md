ATM Service – Spring Boot + JUnit + Mockito + GitHub Actions <br>
This project is a small but well‑structured Spring Boot application that simulates an ATM system. <br>
It demonstrates clean service‑layer logic, custom exception handling, component‑based design, and professional unit testing using JUnit 5 and Mockito. <br>
A full CI pipeline using GitHub Actions and Playwright is also included. <br>
This project is ideal for showcasing backend engineering, testing practices, and CI/CD skills. <br>
